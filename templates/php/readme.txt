To run locally.

1) Copy this php folder underneath the htdocs folder in a php enabled instance of Apache.
2) Rename this the php folder however you like.  For example "webRockeXDemo"
3) Launch Apache 
4) Navigate in browser to 

http://localhost:80/webRockeXDemo/welcome.php

5) Almost all webrocketX functionality is demonstrated in this application.  It aint pretty but it gives you examples.
6) Look in demo.js for your js examples of using the Webapi
